JQ操作cookie

  1. 1.jquery.cookie.js的配置

    • 首先引入jQuery库文件，在后面引入包含jquery.cookie.js的库文件
    • jquery.cookie.js插件地址 <script type="text/javascript" src="jquery.js"></script>  <script type="text/javascript" src="jquery.cookie.js"></script>

  2. 2.使用方法

    ### Get a cookie ``` $.cookie("get", { name: 'login' }); ``` ### Set a cookie (duration in days): ``` $.cookie("set", { duration: 365, name: 'login', value: 'John' }); ``` ### Delete a cookie ``` $.cookie("delete", { name: 'login' }); ``` ### Verify if a cookie exist, return `true` or `false` ``` $.cookie("exist", { name: 'login' }); ```
